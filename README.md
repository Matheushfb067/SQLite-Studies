# 📚 SQLite Studies

Repository created to store **examples, tests, and practical exercises using SQLite** during my database studies.

The goal of this repository is to practice fundamental SQL commands, understand the structure of relational databases, and build a solid foundation for use in larger projects.

---

# 🗂 Project Structure

```
SQLite/
│
├── database/
│   └── .sqlite database files generated from scripts
│
├── scripts/
│   └── .sql files containing SQL commands
│
├── docs/
│   └── notes and concepts related to the studied topics
│
├── .gitignore
└── README.md
```

### 📁 database

Contains the SQLite database files used in the examples.

### 📁 scripts

Contains SQL scripts used for:

* table creation
* data insertion
* queries
* data modification and deletion

### 📁 docs

Contains theoretical notes used during the learning process.

# 🧠 Studied Topics

During the studies in this repository, the following topics will be covered:

* **SQLite Database**
  Environment, tools, and SQL

* **SQLite Fundamentals**
  CREATE TABLE, INSERT, DELETE, SELECT, UPDATE

* **Schema**
  Data Types, Strict Mode, Primary Key, Foreign Key, Constraints, and Data Modeling

* **Select**
  Operators, LIKE, BETWEEN, IN, Functions, GROUP, ORDER, Subquery, and JOIN

* **Performance**
  Index, Trigger, View, and PRAGMA

* **More SQLite**
  Transactions, ALTER TABLE, CLI, JSON, Upserts, and additional resources

* **LMS Project**
  Complete creation of a table for a Learning Management System (LMS)

# ▶ How to Run the Scripts

1. Open the project in Visual Studio Code

2. Install a SQLite extension

3. Open a `.sql` file

4. Run the commands using:

```
Shift + Enter
```

or by clicking the **Run Query** button.
